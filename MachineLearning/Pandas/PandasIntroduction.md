# What is Pandas
- The name "pandas" has a reference to both "panel data" and "python data analysis" and was created by Wes Mckinney in 2008
- It is a python library used for working with data sets
- It has functions for analyzing, cleaning, exploring, and manipulating data
- Read and write data structures and different formats: csv, JSON, ZIP and so on

**Pandas Data Structure**
- Series
    - One-Dimensional labeled arrays pd.Series
- DataFrames
    - Two-Dimensional data structure with columns, much like a table
- Panel
    - It is a 3D container of data


**How to install Pandas?**
```text
pip install pandas
```

**How to import it?**
```text

```
