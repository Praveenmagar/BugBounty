# 6-Month Red Team Learning Plan (4 Hours Per Day)
## Month 1 – Networking, Linux, and Reconnaissance
    - Understand TCP/IP, OSI Model, DNS, HTTP/HTTPS, and ports.
    - Practice Linux commands and Bash basics.
    - Tools to practice: Nmap, Wireshark, tcpdump, netcat.
    - Hands‑on: Scan networks, analyze packets, and enumerate services.
    - Platforms: TryHackMe – Pre Security and Network Fundamentals rooms.

## Month 2 – Web Application Hacking
    - Learn HTTP request/response structure and authentication mechanisms.
    - Study OWASP Top 10 vulnerabilities.
    - Practice: SQL Injection, XSS, CSRF, IDOR, file upload vulnerabilities.
    - Tools: Burp Suite, ffuf, dirsearch, gobuster.
    - Practice platforms: PortSwigger Web Security Academy, OWASP Juice Shop, DVWA.

## Month 3 – Bug Bounty Recon and Automation
    - Master subdomain enumeration and reconnaissance workflows.
    - Tools: subfinder, amass, httpx, katana, nuclei, shuffledns.
    - Build a recon automation workflow pipeline.
    - Create a project: Recon Automation Tool with enumeration and vulnerability scanning.

## Month 4 – Active Directory Attacks
    - Learn Active Directory structure and Kerberos authentication.
    - Practice attacks: Kerberoasting, Pass‑the‑Hash, Golden Ticket, lateral movement.
    - Tools: BloodHound, Mimikatz, CrackMapExec, Impacket.
    - Practice platforms: TryHackMe Active Directory labs, HackTheBox AD labs.

## Month 5 – Exploitation and Privilege Escalation
    - Learn Linux privilege escalation techniques: SUID, cron jobs, weak permissions.
    - Learn Windows privilege escalation techniques: DLL hijacking, weak services.
    - Tools: linpeas, pspy, winPEAS, PowerUp.
    - Practice platforms: HackTheBox and TryHackMe privilege escalation labs.
## Month 6 – Red Team Operations
    - Learn post‑exploitation techniques such as persistence, pivoting, and data exfiltration.
    - Understand command‑and‑control frameworks.
    - Tools: Sliver, Cobalt Strike, Empire, Metasploit.
    - Practice social engineering and phishing simulations.
    - Build projects: Reverse Shell Generator improvement and custom C2 server.

## Weekly Practice Recommendation
    - Complete 3 TryHackMe rooms each week.
    - Solve 2 HackTheBox machines each week.
    - Write at least 1 vulnerability write‑up weekly.
