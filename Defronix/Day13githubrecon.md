# Searching sensitive data from GitHub
- Login to GitHub
- Go to search bar
- Search(GitHub dorking)
	```
    "defronix.com" password
    ```
	```
    "defronix.com" api_key
    ```
	- Here `defronix.com` is called keyword and `api_key` or `password` is dork
- There are two ways to perform GitHub Recon
	1. Automated: Different tools are used
	2. Manual: Search one by one by manually
	- `Between these two manual is best`
